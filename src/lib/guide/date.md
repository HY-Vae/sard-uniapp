## 日期格式特殊符号

根据传入的带有特殊符号的字符串获取格式化的日期。

|      | 描述             |
| ---- | ---------------- |
| YYYY | 四位年份         |
| YY   | 两位年份         |
| MM   | 两位月份         |
| M    | 一位月份         |
| DD   | 两位日期         |
| D    | 一位日期         |
| HH   | 两位二十四制小时 |
| H    | 一位二十四制小时 |
| hh   | 两位十二制小时   |
| h    | 一位十二制小时   |
| mm   | 两位分钟         |
| m    | 一位分钟         |
| ss   | 两位秒数         |
| s    | 一位秒数         |
| SSS  | 毫秒数           |
